General Purpose - This Program is used for distributing cases as a batch for Debt Recovery Company(DRC)


(actionManipulation/Batch_Approved_Case_Process.py)
      *contains the logic for the program


'''Program Flow'''

1.Find the system task with Template_Task_Id = 29 and task_status = "Open".

2.Get the case_distribution_batch_id from parameters.

3.Retrieve the corresponding approver from Template_forwarded_approver.

4.Check if the approver_type is "DRC_Distribution".

5.Fetch case distribution details from Template_case_distribution_drc_details using case_distribution_batch_id.

6.For each case in the batch:

            Check if proceed_on is None.

7.If yes, call the API \Case_Distribution_To_DRC with case_id.

8.After processing, update:

            set Task status to "Complete".

            Task description with the number of errors that occurred.



    Mongo Collections:
    -System_tasks:                             -Read only
    -tmp_forwarded_approver :                  -Read only
    -Temp_Case_Distribution_DRC_Details:       -Read only
